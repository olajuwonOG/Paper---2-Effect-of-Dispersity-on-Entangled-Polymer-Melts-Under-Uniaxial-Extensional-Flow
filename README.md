Paper 2 - Dispersity Effects on Entangled Polymer Melts Under Uniaxial Extensional Flow
Codes (Lammps and Python) for Impact of Dispersity on the Relaxation of Entangled Polymers from Nonequilibrium Molecular Dynamics Simulations paper in the SAMPATH research group

◦ mkinput_monodisperse.py – Python code to generate initial configurations of monodisperse melt

◦ mkinput_polydisperse.py – Python script to generate initial configurations of polydisperse melt

◦ uefex.in – LAMMPS input script for deformation


◦ radiusOfGyration_Deform.py – Radius of gyration calculation for deforming chains and relaxing chains

◦ tube_radius.py – calculates the tube radius evolution for selected test chains

◦ stress_ratio.py – calculates the stretch ratio evolution for selected test chains

◦ ratio_of_tube_elongation.py – calculates the tube elongation for selected test chains

◦ orientation_order_n.py – Hermans orientation parameter calculation for test chains

◦ Gprime_Gprimeprime.py – Storage and loss modulus calculations from equilibrium stress data

*** We use the Z1+ code to calculate relevant entanglement properties (contour length of the primitive path Lpp, entanglement length Ne, and Number of entanglement Z ) --- https://www.sciencedirect.com/science/article/pii/S0010465522002867#se0020
